# Endpoint Triage Playbook

## Objective

Perform rapid investigation of suspicious endpoint activity using OSQuery.

## Steps

1. Identify suspicious processes
2. Review outbound network activity
3. Check persistence mechanisms
4. Validate user accounts
5. Determine containment actions
6. Document findings

## Common Indicators

* Processes running from temporary folders
* Unexpected listening ports
* Unknown user accounts
* Persistent startup entries
* Abnormal outbound connections
